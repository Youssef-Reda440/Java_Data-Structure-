## üìñ Description
This project contains a custom implementation of a **Queue Data Structure using an Array** in **Java**.  
The queue follows the **FIFO (First In First Out)** principle and supports basic queue operations such as enqueue, dequeue, peek, and display, without using built-in collection libraries.

---

## üõ† Features
- Enqueue (Insert elements into the queue)
- Dequeue (Remove elements from the queue)
- Peek (Get the front element)
- Check if the queue is empty
- Check if the queue is full
- Display all queue elements
